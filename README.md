# 轻量日记

功能：

- 多伦多时间日期
- 心情、天气、饮水
- 睡眠、饮食、阅读、运动
- 流水账，支持心情 / 宝宝 / 日常 / 身体 emoji 快捷选择，也支持自定义 emoji 或文字标签
- 灵感、新知
- 日记库搜索
- JSON 备份 / 导入，CSV 导出
- 本地自动保存

注意：数据保存在当前设备当前浏览器的本地存储，不是云同步。换设备前请先导出 JSON，再在新设备导入。

## iPhone 使用

这个版本是 iOS PWA，不需要 Mac 或 Xcode。

1. 解压 ZIP。
2. 把文件夹里的内容上传到 GitHub Pages / Netlify / Cloudflare Pages。
3. 用 iPhone Safari 打开网址。
4. 点分享按钮。
5. 选择“添加到主屏幕”。

添加后会像普通 App 一样从主屏幕打开。
