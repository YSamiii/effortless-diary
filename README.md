# 轻量日记 iPhone 无 Mac 安装版（PWA）

这是一个可安装到 iPhone 主屏幕的网页版 App。没有 Mac 时，这是最省事的方案。

## 安装方式

1. 把本文件夹上传到任意静态网站托管服务，例如 GitHub Pages、Netlify、Vercel 或 Cloudflare Pages。
2. 用 iPhone 的 Safari 打开托管后的网址。
3. 点 Safari 底部分享按钮。
4. 选择“添加到主屏幕”。
5. 主屏幕会出现“轻量日记”图标，之后像普通 App 一样打开。

## 数据保存

数据保存在 iPhone Safari / 主屏幕 Web App 的本地存储中。
建议定期在 App 里使用“导出 JSON”备份。

## 注意

- 需要通过 HTTPS 网址打开，离线缓存和主屏幕体验才比较稳定。
- 这个版本不需要 Mac，也不需要 Xcode。
- 当前不含 iCloud 同步。
